Hex Explained

#RRGGBB

Hex colors allow designers and developers to agree on the exact color they're referring to. RR (red), GG (green) and BB (blue) are hexadecimal values between 00 and ff (same as decimal 0-255). For example, #ff0000 is displayed as red, because red is set to its highest value (ff), and the other two (green and blue) are set to 00.
